Ride Sharing System

Python Folium Geopy

The project aimed to develop a ride sharing route optimization system using graph based algorithms to determine the best pickup and drop sequences for multiple users based on real world locations.

Algorithms Implemented are BFS for shortest path between two points, Prim's Algorithm for generating a Minimum Spanning Tree and Travelling Salesman Problem for finding the most efficient pickup order among multiple starting points.

Meteorite Landing Analysis

Python Streamlit Scikit-learn

The project analyzes a real world meteorite landing dataset using predictive and statistical methods. The goal is to extract meaningful patterns in meteorite characteristics and forecast future trends.

PCA revealed that 3 components explain over 80% of the variance. Clustering showed 3 distinct groups, with Cluster 2 being more scattered.
